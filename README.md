cbplib - C-Berry Python Lib
===========================
Python Lib für das C-Berry TFT-Display.

Vielen Dank an das deutsche Raspberry Pi Forum http://www.forum-raspberrypi.de und die Benutzer:
* @DerHagen - für die erweiterung des C-Berry C Codes
* @meigrafd & @barni7 - für die Idee das Display mit Python über PyGame anzusprechen.

1. Install:
---------------------------
$ sudo apt-get install python-dev
$ sudo python setup.py install

2. Test:
---------------------------
$ sudo python test.py
